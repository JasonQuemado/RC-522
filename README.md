When isntalling the rc-522 libraries you have to install:
```
echo pip install mfrc522
```
instead of:
```
pip install pi-rc522
```


| RC-522 pins | Raspberry Pi pins |
| --- | --- |
| SDA | GPIO 24 |
| SCK | GPIO 23 |
| MOSI | GPIO 19 |
| MISO | GPIO 21 |
| IRQ |  |
| GND | GPIO 6 |
| RST | GPIO 22 |
| 3.3V | GPIO 1 |
